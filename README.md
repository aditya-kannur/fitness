# 💪 Simple Fitness App (Flutter)

A beginner-friendly Flutter app to help users perform and track basic fitness exercises like pushups, situps, squats, lunges, and more. The app includes individual workout screens, progress tracking, timers, and a clean UI.

---

## 📱 Features

- 🏠 Home screen with workout options  
- ✅ Individual workout pages (Pushups, Situps, Squats, Lunges, etc.)  
- ⏱ Repetition counters and optional timers  
- 📊 Progress tracking stored locally  
- 📅 Plan screen with weekly progress overview    
- 🎨 Clean, modern, and responsive UI  

---


## 🛠 How to Run the Project

1. **Install Flutter**  
   https://docs.flutter.dev/get-started/install

2. **Clone this repo**  
   ```bash
   git clone https://github.com/your-username/fitness-app-flutter.git
   cd fitness-app-flutter
   
3. **Install dependencies**
    ```flutter pub get```

4. **Run the app** 
    ```flutter run```
